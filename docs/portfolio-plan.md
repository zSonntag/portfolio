# Portfolio Plan

## Goal

Create a modern and responsive developer portfolio that presents my
experience, skills, learning goals and contact information.

## Target Audience

- Software development recruiters
- Junior developer hiring managers
- Technology companies in Switzerland

## Sections

1. Header
2. Hero
3. About
4. Experience
5. Skills
6. Contact
7. Footer

## Version 1 Requirements

- Responsive layout
- Dark and light theme
- Semantic HTML
- Accessible navigation
- Angular standalone components
- SCSS styling
- No backend
- No unfinished project placeholders

## Design Direction

- Modern and professional
- Dark anthracite background
- Blue accent colour
- Clear typography
- Subtle animations
- No skill percentage bars
